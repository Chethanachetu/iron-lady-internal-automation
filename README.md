# Internal Business Automation Solution – Iron Lady

## Problem Identified
Iron Lady’s internal team manages applicant data manually using spreadsheets.
This causes time consumption, data duplication, and errors.

## Solution
A web-based internal automation application that helps manage applicants efficiently.

## Workflow
1. Add applicant details
2. View applicant list
3. Update applicant information
4. Delete applicant records

## Features
- Create, Read, Update, Delete (CRUD)
- Simple user interface
- SQLite database
- Flask backend

## AI Usage
AI was used to design the workflow, generate code, and debug errors.

## Tech Stack
- Python
- Flask
- SQLite
- HTML / CSS
